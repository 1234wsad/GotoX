# 使用
- Windows 用户：
    - 已安装 Python2，直接双击 `uploader.py` 运行即可；
    - 或者访问[发布页面](https://github.com/SeaHOH/GotoX/releases)下载包含便携 Python2 的发布版，然后双击 `uploader.bat` 运行。
- Linux/Mac 用户：
    - 直接运行命令 `python uploader.py` 即可。
